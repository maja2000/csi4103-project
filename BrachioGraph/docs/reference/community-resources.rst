Community resources
===================

BrachioGraphs benefits from community contributions.


Software
--------

`brachio.me <https://brachio.me>`_, a web version of the linedraw software used to vectorise images, by Rob Bricheno.


Hardware
--------

<https://github.com/ringo2k>`_
